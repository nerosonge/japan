# 日本語クイズ | 일본어 단어 퀴즈

일본어 단어를 퀴즈로 연습할 수 있는 웹사이트입니다.

## 사용 방법

1. `index.html` 파일을 브라우저에서 열어 실행합니다.
2. **퀴즈 모드**를 선택합니다.
   - **일본어 → 뜻**: 일본어 단어를 보고 뜻(한국어)을 고릅니다.
   - **뜻 → 일본어**: 뜻을 보고 맞는 일본어를 고릅니다.
3. **퀴즈 시작**을 누르면 10문제가 출제됩니다.
4. 4지선다 중에서 정답을 클릭하면 다음 문제로 넘어갑니다.
5. 마지막에 맞힌 개수와 함께 결과가 표시됩니다. **다시 하기**로 처음 화면으로 돌아갈 수 있습니다.

## 포함된 단어

- 기본 한자어: 水(물), 火(불), 人(사람), 山(산), 川(강), 木(나무) 등
- 일상 단어: 友達(친구), 先生(선생님), 名前(이름) 등
- 동사: 食べる(먹다), 飲む(마시다), 行く(가다), 来る(오다) 등
- 인사·표현: ありがとう(감사합니다), こんにちは(안녕하세요) 등

브라우저만 있으면 별도 설치 없이 사용할 수 있습니다.
